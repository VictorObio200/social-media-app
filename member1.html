Victor's contribution

In this project, I was responsible for developing the core functionality of the social media web application.

I worked on connecting the frontend to the Supabase database and ensuring that data could be stored and retrieved correctly.

I implemented the Create, Read, and Delete features (CRUD functionality), allowing users to add posts, view posts, and delete them.

I also created the database structure using SQL, including the users and posts tables, and set up the relationship between them using foreign keys.

In addition, I helped debug issues such as connection errors, missing tables, and data insertion problems to ensure the application worked smoothly.
